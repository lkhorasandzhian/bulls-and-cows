------------------|Game "Bulls and Cows"|--------------------
-------------------------------------------------- ------------
Rules:
     The program guesses the player a random four-digit
a number without leading zeros and without repeating digits. Player task
is to guess the number given by the program.

     When starting the game, the player immediately needs to enter his number, without
leading zeros and no repeating digits. With CORRECT input
the game compares the numbers of the hidden random number with the number,
entered player. If the input is incorrect, the game asks the player to enter
number again until the player enters the correct number.

     If one of the digits of the first number is equal to one of the digits
second number, and these digits are located in the same positions in
four-digit numbers, then the game credits the player with this result
as "1 bull". At the exit, the player receives the total number of bulls received.

     If one of the digits of the first number is equal to one of the digits
second number, but these numbers are located in different positions in
four-digit numbers, then the game credits the player with this result
like "1 cow". At the exit, the player receives the total number of received cows.

     After the correct number entered by the user, the display is displayed
number of cows and bulls received by the user for the current attempt to guess
hidden number.

     If the user got 4 bulls and 0 cows (meaning that
he guessed the whole number as a whole), then the game reports a successfully guessed number and
offers to either start a new game (with a new randomly generated number)
by pressing the "R" key, or end the game by pressing any OTHER key.
     If the user has NOT received 4 bulls and 0 cows, then the game reports that the number
NOT guessed. The next line, the game prompts you to either press any key to
continue the current part, or press the "Esc" key to end the current part.
Already then, in the next line, the game offers either to start a new game
(with a new randomly generated number) by pressing the "R" key, or end the game
by pressing any OTHER key.

------------------|Игра "Быки и Коровы"|-------------------
--------------------------------------------------------------
Правила:
     Программа загадывает игроку случайное четырёхзначное
число без ведущих нулей и без повторяющихся цифр. Задача игрока
заключается в том, чтобы угадать число, загаданное программой.

     При запуске игры игроку сразу же требуется ввести своё число, без
ведущих нулей и без повторяющихся цифр. При КОРРЕКТНОМ вводе
игра сверяет по цифрам загаданное радномное числос числом,
введёным игроком. При НЕкорректном вводе игра просит игрока ввести
число заново до тех пор, пока игрок не введёт корректное число.

     Если одна из цифр первого числа оказалась равной одной из цифр
второго числа, и эти цифры располагаются на одинаковых позициях в
своих четырёхзначных числах, то игра засчитывает игроку этот результат
как "1 бык". На выход игрок получает общее кол-во полученных быков.

     Если одна из цифр первого числа оказалась равной одной из цифр
второго числа, но эти цифры располагаются на разных позициях в
своих четырёхзначных числах, то игра засчитывает игроку этот результат
как "1 корова". На выход игрок получает общее кол-во полученных коров.

     После введённого пользователем корректного числа происходит вывод на экран
кол-ва коров и быков, полученное пользователем за текущую попытку угадать
загаданное число.

     Если пользователь получил 4 быка и 0 коров (это означает, что
он угадал всё число в целом), то игра сообщает об успешно угаданном числе и
предлагает либо начать новую партию (с новым рандомно сгенерированным числом)
нажатием клавиши "R", либо завершить игру нажатием любой ДРУГОЙ клавиши.
     Если пользователь НЕ получил 4 быка и 0 коров, то игра сообщает о том, что число
НЕ угадано. Следующей строкой игра предлагает либо нажать любую клавишу для
продолжения текущей партии, либо нажать клавишу "Esc" для завершения текущей парти.
Уже затем, следующей строкой игра предлагает либоначать новую партию
(с новым рандомно сгенерированным числом) нажатием клавиши "R", либо завершить игру
нажатием любой ДРУГОЙ клавиши.
